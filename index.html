About

Welcome to Game of Life - a JavaScript adaptation of Conway's Game of Life. Game of Life can be described as a mathematics-based game, where cells within the game will either live on or die in the next generation according to the number of neighbours that are alive.
Rules:
  • Any live cell with fewer than two live neighbours dies, as if caused by under-population
  • Any live cell with two or three live neighbours lives on to the next generation.
  • Any live cell with more than three live neighbours dies, as if by overcrowding.
  • Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.
Following these rules, it is possible to create repeating patterns or constructors/factories. Try it yourself, or search online for defined examples.

Help

In the Play/Edit options:
  • Play Mode: Runs the current set up of the game, proceeding through the generations.
  • Edit Mode: Stops the game, and allows you to click on the grid, setting up alive or dead cells.
In the Settings options:
  • Rows & Columns: Changes the dimensions of the playable space for the game (WARNING: Larger dimensions adversely affects performance).
